Simply Boilerplate
=======================
The boilerplate for WordPress project. It integrates [WordPress](https://wordpress.org/), [Simply framework](https://github.com/Amorfx/simply-framework) and [Simply maker](https://github.com/Amorfx/simply-maker).

## Why using Simply ?
- Use .env and not directly wp-config.php file
- Save time with the Simply CLI
- Install directly WordPress with Composer
- Manage your plugins/themes and more with composer
